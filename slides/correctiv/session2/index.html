It's NOT:

  • A conversation with a similarly thinking entity
  • It will present bullshit confidently
  • Don't be fooled! Keep thinking!

Instead:

  • Context building
  • Transformation

Take it step by step

  • start with exploration and summary stats
  • drill into what's interesting: correlations, outliers
  • combine with other data
  • make dashboards
  • test the output against the source data

If you are unsure about the approach

  • Ask about a good approach
  • Do your analysis as a new iteration — refrain from introducing noise to your context

Always verify

  • LLMs fabricate plausible numbers
  • Check every stat against the source
  • Ask it to cite the rows or cells it used

Data privacy

  • Don't paste sensitive data into hosted chat UIs
  • Leaked docs, sources, personal data → local agents only
  • Check the tool's data-retention settings

Different from a chat UI

  • Reads your files directly
  • Runs code on your machine
  • Keeps working while you watch
  • You need a subscription
